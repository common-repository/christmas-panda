== Description ==

Christmas decorations plugin for WordPress. Decorate your WordPress website with Christmas trees, Santa, snowfall or just display a pop-up to remember your visitors that it's Christmas.

https://www.youtube.com/watch?v=AXo0mqt1ZNg

Features:

*   Christmas Pop-ups
*   Snowfall animations
*   Christmas decorations

Admin Features:

*	Manage decorations
* 	Enable/Disable snowfall
* 	Manage popups

For further information and instructions please see the <a href="https://pixolette.com/wordpress-plugins/christmas-panda/">plugin page</a>

== Installation ==

The quickest method for installing the Christmas Panda is:

1. Visit Plugins -> Upload Plugin -> Choose File "christmas-panda.zip"
2. Click "Install Now"
3. Finally click "Activate Plugin"

If you would prefer to do things manually then follow these instructions:

1. Upload the `halloween-panda` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
